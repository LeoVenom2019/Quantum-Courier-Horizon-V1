# QCH Balance Guidelines

## Economy

- Compare production, storage, costs, and rewards across adjacent tiers.
- Check whether reducer formulas and UI descriptions agree.
- Treat very large multipliers as progression milestones that should be intentional.

## Combat

- Compare enemy HP/damage/reward against player upgrade paths.
- Check cooldowns and spawn rates together; either can dominate difficulty.
- Boss jumps should feel like route milestones, not accidental numeric spikes.

## Missions

- Mission rewards should match effort and unlock timing.
- Avoid objectives that require state no reducer can currently advance.

## Persistence

- When adding a new numeric field, check default state, loading/migration behavior, and reset paths.
